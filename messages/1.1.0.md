# Spartan Color Scheme 1.1.0

## What's New

- Light variant for bright environments
- Legacy .tmTheme format for Sublime Text < Build 3149
- GitGutter support with colored gutter marks
- BracketHighlighter support with per-bracket-type colors
- SublimeLinter 4 support for warning and error marks
- Twig and Handlebars template engine highlighting
- LESS/Sass preprocessor variable and mixin highlighting
- PHP namespace highlighting with 4 dedicated scopes
- Shell/Bash function, variable, and builtin highlighting
- CoffeeScript and Makefile language support
- Expanded palette from 19 to 38 color variables
- Added 12 missing editor globals for full integration

## Activation

Dark: `Preferences > Color Scheme > Spartan`
Light: `Preferences > Color Scheme > Spartan Light`
