# Spartan Color Scheme

Thank you for installing Spartan Color Scheme.

## Activation

### Dark (default)

Select `Preferences > Color Scheme > Spartan` from the main menu.

### Light

Select `Preferences > Color Scheme > Spartan Light` from the main menu.

### Sublime Text < Build 3149

Select `Preferences > Settings` from the main menu and set:

```json
"color_scheme": "Packages/Spartan Color Scheme/Spartan.tmTheme"
```
